Name

lcb_destroy_io_ops — Destroy io ops instance

Synopsis

cc [ flag … ] file… -lcouchbase [ library … ]

#include <libcouchbase/couchbase.h>
lcb_error_t lcb_destroy_io_ops(lcb_io_opt_t op);

DESCRIPTION

lcb_destroy_io_ops() is used to release all allocated resources and invalidate an IO ops instance.

RETURN VALUES

lcb_create() returns the LCB_SUCCESS on success, or a specific error code upon failure. See lcb_strerror(3couchbase) for more information.
