Name

lcb_destroy — Destroy an instance

Synopsis

cc [ flag … ] file… -lcouchbase [ library … ]

#include <libcouchbase/couchbase.h>
void lcb_destroy(lcb_t instance);

DESCRIPTION

lcb_destroy() release all resources allocated by the instance and invalidates the instance. Using a destroyed instance will most likely cause your application to crash.
